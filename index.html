Terrain Simulator

Project Overview:
Terrain Simulator generates realistic 3D terrain with immersive user interaction. Users can explore the environment using "drive" and "fly" modes for dynamic navigation. The simulator supports rainfall-based erosion effect, which is controled by users, and visualizes elevation through color-based height mapping.

3D Model Generator

Project Overview:
3D Model Generator focuses on implementing texture mapping in a 3D rendering pipeline. It allows users to project custom images onto any 3D model surface. The system handles coordinate mapping and interpolation to ensure accurate texture placement.

Collision Simulator

Project Overview:
This project simulates elastic collisions between spheres based on the principles of kinetic energy conservation and Newtonian mechanics. The motion and collision responses are computed using Euler’s method for numerical integration.
